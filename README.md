# SupremeExpansion 至尊研究院

至尊研究院是一个 Slimefun 附属，添加了100多种全新的资源，可以让你合成强力的新物品、武器、工具与防具。你可以使用钛合金、金合金、精金、钍来制作魔法、稀有、史诗、传说、至尊等级的物品。该附属还增加了12种全新的发电机，5种电容，还有更多...
